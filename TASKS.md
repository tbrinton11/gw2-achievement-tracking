# Master Task List

All tasks across the three tracked achievements, sorted from quickest wins to longest grinds.

---

## Quick Wins (< 30 min each)

- [ ] **Buy remaining S3 portal scrolls** (Gizmos) — Visit Bloodstone Fen, Ember Bay, Lake Doric, Draconis Mons, Siren's Landing vendors. 5 scrolls, just costs map currency.
- [ ] **Buy remaining S4 portal scrolls** (Gizmos) — Visit Domain of Istan, Sandswept Isles, Domain of Kourna, Jahai Bluffs, Thunderhead Peaks vendors. 5 scrolls.
- [ ] **Buy remaining IBS portal scrolls** (Gizmos) — Visit Grothmar Valley, Bjora Marches, Drizzlewood Coast vendors. 3 scrolls.
- [ ] **Buy Lion's Arch Teleportation Device** (Gizmos) — Zotturrikk in Hooligan's Route, Lion's Arch.
- [ ] **Buy Hall of Monuments Portal Stone** (Gizmos) — Sarettokk in Hooligan's Route, Lion's Arch. Requires GW1 HoM link.
- [ ] **Check "Poster Child" achievement status** (Gizmos) — If close to done, finish it. Otherwise add scroll to S4 Portal Tome.
- [ ] **Check "Fallout" achievement status** (Gizmos) — May already have prerequisites.
- [ ] **Check Griffon mount status** (Gizmos) — If already unlocked, the gizmo may auto-register by having the item.

## Efficiency Combos

These tasks can be done simultaneously to save time:

1. **S3 Return episodes + S3 portal scroll purchases** — Buy the scroll while you're on the map
2. **S4 Return episodes + S4 portal scroll purchases** — Same idea
3. **Return to Drizzlewood Coast + Otter farming + IBS portal scroll purchase** — Triple value from one map
4. **Return to Bjora Marches + Significant Otter prerequisites** — Both in Bjora Marches
5. **Lion's Arch vendor shopping trip** — Buy LA Teleportation Device + HoM Portal Stone in one visit
