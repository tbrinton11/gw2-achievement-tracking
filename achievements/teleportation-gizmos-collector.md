## Strategy Notes

- **Quick wins first:** Buy all remaining S3/S4/IBS vendor scrolls in one session — just teleport to each map and visit the vendor
- **Combine with Seasons of the Dragons:** When doing Return episodes, buy the map's portal scroll while you're there
- **Mystic Coins:** Start saving/buying now — you need 350 for all city scrolls. Weekly Mystic Clover recipe and login rewards help
- **Festival Frequenter** is calendar-gated — participate in each festival as it comes up (Lunar New Year, Super Adventure Box, Dragon Bash, Festival of the Four Winds, Halloween, Wintersday)
- **Chuka and Champawat IV** is a legendary collection — only pursue if you're also interested in the legendary shortbow
